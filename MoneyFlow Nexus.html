Frontend Stack (Client-Side)

  • React 18.2.0 UI Library (ใช้ผ่าน UMD แบบไม่ต้อง Build)
  • JSX / JavaScript ES6+ เขียนด้วย Babel Standalone ในเบราว์เซอร์
  • CSS3 (Custom Properties) CSS Variables สำหรับ Theme System ทั้ง Dark/Light Mode
  • Chart.js 4.4.0 สร้างกราฟแท่งและกราฟวงกลมแบบ Interactive
  • Google Fonts Noto Sans Thai, Rajdhani, JetBrains Mono
  • Font Awesome 6.5.1 ชุดไอคอนกว่า 2,000+ ตัว

Backend Stack (Server-Side)

  • Google Sheets ฐานข้อมูล NoSQL-like เก็บข้อมูลในรูปแบบตาราง
  • Google Apps Script JavaScript-based Serverless Backend
  • RESTful API HTTP GET/POST ผ่าน Web App Deployment
  • JSON Data Exchange รับ-ส่งข้อมูลในรูปแบบ JSON
  • GitHub Pages Hosting แบบ Static Site ฟรี

คู่มือการติดตั้งด้วยตนเอง (สำหรับเจ้าของโค้ด)

คู่มือนี้จะพาคุณติดตั้งระบบทั้งหมดภายใน 10-15 นาที โดยใช้ทรัพยากรฟรีจาก Google และ GitHub

1

📁 สร้าง Google Sheets และ Apps Script

1. ไปที่ sheets.new สร้างชีตใหม่
2. เปลี่ยนชื่อ Sheet เป็น Transactions, Wallets, ExpenseCats, IncomeCats, Settings (ตามชื่อ Tab ด้านล่าง)
3. ไปที่ ส่วนขยาย (Extensions) > Apps Script

// วางโค้ดนี้ลงใน Apps Script Editor
function doGet(e) { return handleRequest(e); }
function doPost(e) { return handleRequest(e); }

function handleRequest(e) {
  try {
    const sheet = SpreadsheetApp.getActiveSpreadsheet();
    // จัดการ action ต่างๆ
    return ContentService.createTextOutput(JSON.stringify({ success: true, data: [] }))
      .setMimeType(ContentService.MimeType.JSON);
  } catch(error) {
    return ContentService.createTextOutput(JSON.stringify({ success: false, error: error.toString() }))
      .setMimeType(ContentService.MimeType.JSON);
  }
}

4. กด บันทึก > ทำให้ใช้งานได้ (Deploy) > New Deployment > เลือก Web App
5. ตั้งค่า Execute as: Me, Who has access: Anyone (สำคัญ!) > กด Deploy และ คัดลอก URL ที่ได้

2

🔗 เชื่อมต่อ URL ในโค้ด HTML

เปิดไฟล์ index.html (ไฟล์หลักของระบบ) และค้นหาบรรทัด:

const NEXO_API_URL = 'YOUR_SCRIPT_URL_HERE';

นำ URL ที่คัดลอกมาจากขั้นตอนที่ 1 มาใส่แทนที่ YOUR_SCRIPT_URL_HERE (ต้องเก็บเครื่องหมาย ' ไว้ด้วย)

หมายเหตุ: หลังจากเปลี่ยน URL ทุกครั้ง หากมีการแก้ไข Apps Script ต้องกด Deploy > New Version และอัปเดต URL ในโค้ดด้วย

🌍 อัปโหลดขึ้น GitHub Pages

1. สร้าง Repository ใหม่บน GitHub (ชื่อเช่น moneyflow-app)
2. อัปโหลดไฟล์ index.html ของคุณขึ้นไปที่ Repository

📊 ตั้งค่าข้อมูลเริ่มต้น (Initial Setup)

เมื่อเข้าใช้งานครั้งแรก ระบบจะพยายามดึงข้อมูลจาก Sheets ถ้า Sheet ว่างเปล่า ระบบจะใช้ข้อมูลตัวอย่าง (Default Wallets/Categories)
แนะนำให้เปิด Apps Script และรันฟังก์ชัน initSheets() เพื่อสร้าง Header และข้อมูลตัวอย่างโดยอัตโนมัติ

// เพิ่มใน Apps Script แล้วกด Run หนึ่งครั้ง
function initSheets() {
  const ss = SpreadsheetApp.getActiveSpreadsheet();
  
  // สร้าง Transactions sheet
  let sheet = ss.getSheetByName('Transactions') || ss.insertSheet('Transactions');
  sheet.getRange('A1:H1').setValues([['id', 'type', 'amount', 'catId', 'walletId', 'date', 'time', 'note']]);
  
  // สร้าง Wallets sheet พร้อมข้อมูลเริ่มต้น
  sheet = ss.getSheetByName('Wallets') || ss.insertSheet('Wallets');
  sheet.getRange('A1:D1').setValues([['id', 'name', 'icon', 'color']]);
  sheet.getRange('A2:D4').setValues([
    ['w1', 'กระเป๋าหลัก', 'fa-wallet', '#6c63ff'],
    ['w2', 'บัตรเครดิต', 'fa-credit-card', '#8b5cf6'],
    ['w3', 'ออมทรัพย์', 'fa-piggy-bank', '#00d68f']
  ]);
  
  console.log('✅ Initial setup complete!');
}
