คู่มือการติดตั้งด้วยตนเอง (สำหรับเจ้าของโค้ด)

คู่มือนี้จะพาคุณติดตั้งระบบทั้งหมดภายใน 10-15 นาที โดยใช้ทรัพยากรฟรีจาก Google และ GitHub

1

📁 สร้าง Google Sheets และ Apps Script

1. ไปที่ sheets.new สร้างชีตใหม่
2. เปลี่ยนชื่อ Sheet เป็น Transactions, Wallets, ExpenseCats, IncomeCats, Settings (ตามชื่อ Tab ด้านล่าง)
3. ไปที่ ส่วนขยาย (Extensions) > Apps Script

// วางโค้ดนี้ลงใน Apps Script Editor
function doGet(e) { return handleRequest(e); }
function doPost(e) { return handleRequest(e); }
// ... (แนะนำให้ใช้โค้ดจากไฟล์ script ที่คุณเตรียมไว้)

4. กด บันทึก > ทำให้ใช้งานได้ (Deploy) > New Deployment > เลือก Web App
5. ตั้งค่า Execute as: Me, Who has access: Anyone (สำคัญ!) > กด Deploy และ คัดลอก URL ที่ได้

2

🔗 เชื่อมต่อ URL ในโค้ด HTML

เปิดไฟล์ index.html (ไฟล์หลักของระบบ) และค้นหาบรรทัด:

const NEXO_API_URL = 'YOUR_SCRIPT_URL_HERE';

นำ URL ที่คัดลอกมาจากขั้นตอนที่ 1 มาใส่แทนที่ YOUR_SCRIPT_URL_HERE (ต้องเก็บเครื่องหมาย ' ไว้ด้วย)

หมายเหตุ: หลังจากเปลี่ยน URL ทุกครั้ง หากมีการแก้ไข Apps Script ต้องกด Deploy > New Version และอัปเดต URL ในโค้ดด้วย

3

🌍 อัปโหลดขึ้น GitHub Pages

1. สร้าง Repository ใหม่บน GitHub (ชื่อเช่น moneyflow-app)
2. อัปโหลดไฟล์ index.html ของคุณขึ้นไปที่ Repository
3. ไปที่ Settings > Pages
4. ใต้ "Branch" เลือก main หรือ master แล้วกด Save
5. รอประมาณ 1-2 นาที ระบบจะสร้างลิงก์สำหรับเข้าใช้งาน เช่น https://username.github.io/moneyflow-app

ดูตัวอย่างโค้ดบน GitHub
4

📊 ตั้งค่าข้อมูลเริ่มต้น (Initial Setup)

เมื่อเข้าใช้งานครั้งแรก ระบบจะพยายามดึงข้อมูลจาก Sheets ถ้า Sheet ว่างเปล่า ระบบจะใช้ข้อมูลตัวอย่าง (Default Wallets/Categories)
แนะนำให้เปิด Apps Script และรันฟังก์ชัน initSheets() เพื่อสร้าง Header และข้อมูลตัวอย่างโดยอัตโนมัติ

// เพิ่มใน Apps Script แล้วกด Run หนึ่งครั้ง
function initSheets() {
  // สร้าง Header และ Default Data
}
